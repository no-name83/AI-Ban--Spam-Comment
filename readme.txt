== Description ==
This plugin uses the OpenAI GPT-3.5 text-davinci-003 model to analyze and filter comments made on your texts. You can choose to have spam comments either deleted without being added to your database or saved as junk comments. To use the plugin, you just need to enter your OpenAI key.
<h3>REQUIRES:</h3>
 - PHP 7.2 or higher
 - WordPress 5.0.x or higher
